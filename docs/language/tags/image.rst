Images
======

Tags to insert and format images

``@img{...}``
   Insert an image.

   .. index::
      single: tags; @img

   :attributes:

      Depenency attributes

   :examples:

      ::
         
         @img[class=icon]{media/imgs/template.svg}
         @img[tex.width=300 html.width=150 id=figure-1]{media/imgs/simulation.svg}
